# lammps-data-processing

对于开源分子动力学模拟软件`Lammps`导出的大量数据文件，其实是不太适合利用`Excel`或者`Origin`等常规软件来进行数据处理和绘图的。而笔者从2020年正式学习`Lammps`开始的时候就做了利用`Python`进行全流程（后处理+科研绘图）的尝试，也都取得了相关的论文成果，论文中结构示意图以外的数据图都是采用`Python`来进行的。
